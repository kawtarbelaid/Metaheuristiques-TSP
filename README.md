# Metaheuristics for TSP

This project compares the performance of different approaches for solving the Traveling Salesman Problem (TSP), including:

- Greedy heuristic
- Hybrid metaheuristic architecture combining:
  - Genetic Algorithm (GA)
  - Simulated Annealing (SA)
  - Threshold Accepting (TA) with Elite Pool

The comparison is performed on TSPLIB benchmark instances.
